## General info
Just a website for me myself and i.

A personal website built with ASP.NET Core.
The site contains small projects, experiments, football data, weather data, links, notes and other things I want to host myself.

The project originally ran in Azure, but is now also set up to run self-hosted with Docker.

## Tech stack
Tech stack
.NET 8
ASP.NET Core MVC
Razor views
Entity Framework Core
MariaDB
Docker
Docker Compose
nginx
Let’s Encrypt
DuckDNS
